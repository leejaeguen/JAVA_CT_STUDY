```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        long[] squares = new long[n+2];
        squares[0] = 1;
        squares[1] = 1;

        for(int i = 2; i <= n+1; i++) {
            squares[i] = squares[i-2] + squares[i-1];
        }

        long answer = squares[n-1] * 2 + squares[n] * 2;
        System.out.println(answer);
    }
}

```
<br>

1. 정사각형 타일의 길이

- squares 배열 생성 (n+2 크기)
- squares[0], squares[1]을 1로 초기화
- 정사각형 타일의 길이는 아래와 같이 증가함
```agsl
1, 1, 2, 3, 5, 8, 13, 21, ... 
```
- 따라서 squares[i] = squares[i-2] + squares[i]로 n+1번째 까지 배열 채움

<br>

2. 직사각형 둘레 구하기

- 타일의 개수가 n개 일때, 한 변의 길이는 squares[n-1], 한 변의 길이는 squares[n]임
- 각각 두 변씩 있으므로 2씩 곱해서 둘레 구함

<br>

3. 서브 태스트

- int 형이 아닌 long형을 사용
